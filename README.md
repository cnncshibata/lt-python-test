# pytest チュートリアル

pytest の基本的な使い方のサンプルコード

## Usage

### セットアップ

```bash
pipenv install
```

### 実行

```bash
cd [main or coverage]
pytest
```

実行すると htmlcov ディレクトリに html 形式でレポートが出力されます。
